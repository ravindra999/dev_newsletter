Billing Changes

As mentioned in posts to the Shopify Developer Discuss mailing list, the Billing API and Shopify Partner payouts have changed.

The two changes are:

  • Recurring Application Charges will now be paid in advance
  • Partners (i.e. affiliates, theme designers and app developers) will now be paid out twice a month

For more details on these changes, see this post in the mailing list or take a look at http://wiki.shopify.com/Application_Billing

API Updates

Here are the API updates that have gone live in the last two weeks:

  • Metafields: You can now optionally embed metafields into parent objects when reading them.
  • New Webhook Topic: order/partially_fulfilled is fired when an order has some but not all line items fulfilled.

As always, full developer docs can be found here: http://api.shopify.com
