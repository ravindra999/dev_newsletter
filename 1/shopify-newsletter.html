New .NET Adapter

Colin McDonald recently released a new .NET adapter for the Shopify API.

You can grab it from Github, if you're an MS developer you should definitely give it a look. Thanks Colin!

New PHP Adapter Too!

Not content with writing one killer PHP adapter, Sandeep Shetty has producted a new one that supports OAuth 2.0.

Once again, check out the code on Github to get started. Thanks Sandeep!

Line Item Properties

We recently stealth-launched a new Shopify feature: Line Item Properties. These allow merchants to gather extra info from customers on a per-product basis.

This feature is still in its infancy but you can add the functionality to shops right now. An app to automate the process of creating template variants for each product would be a great project to work on.

How to Sell Shopify

There's an article up on the blog aimed at people who use our affiliate program to sell Shopify to their clients. Written by veteran partner Jamie Sutton, it has some great advice on how to get clients interested in using Shopify so that you can reap the benefits.

Read 'How to Sell Shopify to Your Customers' on our tech blog.

API Updates

Here are the API updates that have gone live in July:

  • Fix: notify_customer flag is no longer ignored on fulfillment update
  • Fix: Webhooks are now always removed when app is uninstalled
  • Fix: Setting tracking company to something we don't expect no longer causes a 500 error
  • Update: A list of FedEx shipping codes has been added to the wiki

As always, full developer docs can be found here: http://api.shopify.com
