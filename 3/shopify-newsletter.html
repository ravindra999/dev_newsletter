Hey Developers!

Welcome to the fourth effective issue of the Shopify Developer Newsletter. as usual, previous issues are available on Github

The content of this newsletter is driven by you, our developer community. If you have a Shopify-related project you want to publicize, a new tool you're using to help with your development process, or just want to get more info on a given subject, let us know! We can be reached at apps@shopify.com

Upcoming Changes

Switch to HTTPS

We presently allow API access via HTTP. As of December 6th, we will require all API calls to be made via HTTPS. Please ensure that your app is using the correct protocol before this date.

Removal of Legacy authentication option for new apps

Our options presently for authentication are OAuth 2 and Legacy authentication. We will be removing the option to create new apps with Legacy authentication as of November 22nd. Apps currently using Legacy authentication will still continue to function as expected, but new apps will be required to use OAuth 2.

Recent Updates and General Info

Status code change

We are changing the status code returned when an API client has hit its call limit. Presently, we use 503 Service Unavailable to indicate that a client has reached the call limit. As of November 30th, we will begin using status code 429 Too Many Requests (RFC 6585).

Customer API

We've made some exciting additions to the Customer API - you can now create customer user accounts for a shop via the API! Check out the API docs for details.

Google group - API Announcements

We try to make API announcements as publicly as possible. So, if you don't follow @shopifyapi on Twitter or frequent the forums, worry not - you can keep an eye on our API Announcements Google group.

OAuth 2.0 support added to unofficial Node adapter

The unofficial Node.js Shopify API adapter, Nodify now has added OAuth 2.0 support! The developers, Shopfrogs, have also put together an example project.

If you encounter any API issues or have any questions, you can visit the tech forums or post your questions to Stack Overflow using the 'shopify' keyword. You can also watch the Technology blog and @shopifyapi for updates.
