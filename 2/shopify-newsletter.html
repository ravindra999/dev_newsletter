Checkout API

There are changes afoot in the Shopify API!

Specifically, we're releasing a new Checkout API to track orders between the cart and the finalized, completed, order.

If you have an app that talks to either the Orders or Carts API, make sure to read Lydia's blog post on the subject as you'll need to make changes to your app.

API Updates

Here are the API updates that have gone live in July:

  • New: The Shop object now includes lat and long values for geolocation
  • Update: The 'original' size in the Product Images API now points to a 1024x1024 pixel version of the image. A new 'master' size has been added to serve the image it its true original size
  • New: Product Images can now be updated through the Products API by nesting them under their parent product
  • Fix: Note attributes serialization was broken when Fedex line items was added. This has now been fixed

As always, full developer docs can be found here: http://api.shopify.com
