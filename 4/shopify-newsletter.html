New Features

Changelog

  • Test application charges no longer require billing information to be present on the shop owner's account;
  • Support for multiple tracking numbers has been added to fulfillments.

New Partners Dashboard

With the new dashboard, you'll notice improved performance to feature the important stats you want to monitor, and the ability to export data so you can run your own analysis on the data collected.

What happened to test and affiliate shops? In order to clarify the development process, we introduced development shops. Development shops are free accounts specifically for working on themes, apps, or getting a store set up for a client. They are limited to 100 SKUs and cannot accept payment on orders, but have all the functionality of normal plan. Affiliate shops can be created in 2 ways: through your partner referral link, or by switching to a paid plan on a development shop.

Developer Tools and Guides

Shopify API Console Guide

This little-known tool is great for exploring the API and troubleshooting your API calls. The Shopify console is included in the shopify_api gem, and you can now find a guide on using it on the wiki.

API Troubleshooting guide

Check out this short guide on how to improve your troubleshooting skills when working with the Shopify API. If we've left anything out, leave your feedback in the forums!

iOS Super Debugger

LLDB is the current state of the art for iOS debugging, but it’s clunky and cumbersome and doesn’t work well with objects. It really doesn't feel very different from gdb. It's a solid tool but it requires breakpoints, and although you can integrate with Objective C apps, it's not really built for it. Dealing with objects is cumbersome, and it's hard to see your changes.

This is where Super Debugger comes in. It's a new tool for rapidly exploring your objects in an iOS app whether they're running on an iPhone, iPad, or the iOS Simulator, and it's available today on Github. Check over the included readme to see what it can do in detail.

Sharpify

Shopify community member Andrew Clunis has released a fork of Colin McDonald's original Shopify .NET adapter, titled Sharpify. Sharpify is a type safe, asynchronous, object oriented .NET API wrapper for the Shopify API. You can get the project on GitHub.

Developer Handbook - A Developer's View of the Shopify Platform

Are you looking for a more in-depth view of the Shopify API? Dave Lazar's ebook, A Developer's View of the Shopify Platform is a hidden gem, and a must-read for serious Shopify app developers.

The book is licensed under Creative Commons, and can also be found on GitHub - pull requests are welcome!
